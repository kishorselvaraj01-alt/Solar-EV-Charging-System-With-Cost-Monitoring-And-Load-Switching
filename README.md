# Solar EV Charging System with Cost Monitoring and Load Switching

An ESP32-based solar EV charging system designed for intelligent power management, automatic load switching, and real-time energy cost monitoring.

## Overview
The project utilizes solar energy to charge electric vehicles while supplying power to shop loads through intelligent load switching. An ESP32 microcontroller monitors battery status, energy consumption, and charging cost to optimize power utilization and improve system efficiency.

## Features
- ESP32-Based Control
- Solar-Powered EV Charging
- Automatic Load Switching
- Real-Time Cost Monitoring
- Battery Voltage Monitoring
- Inverter-Based AC Supply
- Efficient Energy Management

## Software
- Arduino IDE
